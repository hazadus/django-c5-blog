# Blog app (Chapter 5)
Tutorial app from ['Django for Beginners' WSV's book](https://learndjango.com/books/).

## Features
- Class-based views -  `CreateView`, `UpdateView`, `DeleteView`.
- Using static CSS files.
- Tests.

## References

## Starting app
```bash
$ source .venv/bin/activate
$ python manage.py runserver
```
